# 高达价格工具 - 启动说明

## 第一步：安装依赖

在项目目录下运行：

```bash
cd gunpla_price_tool
pip install -r requirements.txt
```

## 第二步：运行应用

```bash
python app.py
```

## 第三步：访问应用

打开浏览器访问：http://localhost:5000

## 功能说明

### 已完成的功能

1. ✅ **项目基础搭建**
   - Flask应用框架
   - 数据库模型定义
   - 基础HTML模板

2. ✅ **高达管理**
   - 添加高达信息
   - 查看高达列表
   - 搜索和筛选功能
   - 高达详情页面

3. ✅ **价格显示**
   - 多地区价格显示（日本、美国、中国）
   - "算"数自动计算

4. ✅ **收藏管理**
   - 想要列表
   - 已购买列表
   - 列表间转换功能

5. ✅ **优惠券管理**
   - 添加优惠券
   - 优惠券列表
   - 优惠券分析（推荐最适合的高达）

## 数据库

- 默认使用SQLite数据库
- 数据库文件：`gunpla.db`
- 首次运行会自动创建数据库表

## 注意事项

- 汇率配置在 `config.py` 中，默认 1人民币 = 20日元
- 开发模式下会显示调试信息
- 建议定期备份数据库文件
