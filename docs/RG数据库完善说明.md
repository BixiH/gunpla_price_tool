# RG数据库完善说明

## 当前问题

1. **价格数据错误**：所有价格都是5500日元（价格爬取逻辑有问题）
2. **数据量**：219个模型（根据网页应该是222个，但可能有些是重复的）
3. **子分类识别**：需要改进

## 解决方案

创建一个简单脚本，**只爬取列表信息（不含价格）**：

### 脚本：`完善RG数据库.py`

**功能：**
- ✅ 使用已知的RG URL：`https://acg.78dm.net/ct/341672.html`
- ✅ 只爬取模型列表信息（名称、子分类、编号等）
- ✅ 不爬取价格（价格爬取逻辑需要改进）
- ✅ 自动识别子分类
- ✅ 更新数据库

### 使用方法

```bash
cd gunpla_price_tool
py 完善RG数据库.py
```

### 关于数据量

- 爬取到：219个模型
