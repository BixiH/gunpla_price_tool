# 快速开始指南

## ⚠️ 重要提示

**Flask应用不会自动打开浏览器！** 

运行 `python app.py` 后，您需要**手动打开浏览器**访问应用。

## 📝 完整步骤

### 步骤1：运行应用

在终端（PowerShell或命令提示符）中输入：

```bash
cd gunpla_price_tool
python app.py
```

### 步骤2：查看终端输出

运行后，您应该看到类似这样的输出：

```
 * Serving Flask app 'app'
 * Debug mode: on
WARNING: This is a development server...
 * Running on http://127.0.0.1:5000
Press CTRL+C to quit
```

看到 `Running on http://127.0.0.1:5000` 就说明应用已经启动了！

### 步骤3：打开浏览器

**保持终端窗口打开**，然后：

1. 打开您的浏览器（Chrome、Edge、Firefox都可以）
2. 在地址栏输入：**http://localhost:5000**
3. 按回车键

### 步骤4：使用应用

如果一切正常，您会看到应用的首页！

## 🔍 如果还是看不到页面

### 方法1：检查终端是否有错误

查看终端窗口，如果有红色错误信息，请告诉我错误内容。

### 方法2：尝试直接打开链接

在浏览器中尝试这些地址：
- http://localhost:5000
- http://127.0.0.1:5000

### 方法3：检查端口

如果5000端口被占用，可以修改 `app.py` 的最后一行：

```python
app.run(debug=True, host='0.0.0.0', port=5001)  # 改成5001或其他端口
```

## 🛑 停止应用

在运行应用的终端窗口中，按 `Ctrl+C` 可以停止应用。

## 💡 提示

- 应用运行期间，**终端窗口必须保持打开**
- 不要关闭运行应用的终端窗口
- 如果需要在终端做其他事情，可以打开**新的终端窗口**
