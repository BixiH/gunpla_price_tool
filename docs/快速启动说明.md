# 快速启动说明

## ⚠️ 重要：您需要先进入正确的目录！

### 当前情况

您现在在 `project2-1` 目录中，但 `app.py` 文件在 `gunpla_price_tool` 子目录中。

### ✅ 正确的运行步骤

#### 方法1：先切换目录（推荐）

在终端中输入：

```bash
cd gunpla_price_tool
py app.py
```

#### 方法2：直接从当前目录运行

从 `project2-1` 目录直接运行：

```bash
py gunpla_price_tool/app.py
```

#### 方法3：使用完整路径

```bash
cd C:\Users\bixi\Downloads\project2-1\gunpla_price_tool
py app.py
```

### 📝 完整操作步骤

1. **在终端中输入：**
   ```bash
   cd gunpla_price_tool
   ```

2. **按回车键**，提示符会变成：
   ```
   PS C:\Users\bixi\Downloads\project2-1\gunpla_price_tool>
   ```

3. **输入运行命令：**
   ```bash
   py app.py
   ```

4. **按回车键**，应该看到：
   ```
    * Serving Flask app 'app'
    * Debug mode: on
    * Running on http://127.0.0.1:5000
   Press CTRL+C to quit
   ```

5. **打开浏览器**，访问：http://localhost:5000

### 🎯 如果看到错误

如果提示"找不到文件"或"No such file"，说明您不在正确的目录中。

**解决方法：**
- 确认当前目录（看提示符）
- 使用 `cd gunpla_price_tool` 进入子目录
- 或者使用 `dir` 查看当前目录的文件

### 💡 小技巧

- 使用 `Tab` 键可以自动补全目录名
- 使用 `cd ..` 可以返回上一级目录
- 使用 `dir` 或 `ls` 可以查看当前目录的文件
