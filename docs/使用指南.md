# 高达价格工具 - 使用指南

## 🎉 恭喜！应用已成功运行

现在您可以开始使用这个工具了！

## 📝 第一步：添加测试数据

### 推荐测试数据示例

1. **点击导航栏的"添加高达"按钮**

2. **填写测试数据（示例）：**

   **基本信息：**
   - 中文名称：`RX-78-2 高达`
   - 级别：`MG`
   - 机体编号：`RX-78-2`
   - 所属系列：`机动战士高达`

   **价格信息（可选，但建议至少填写）：**
   - 日本定价：`4500`（日元）
   - 中国市场价格：`250`（人民币）

3. **点击"保存"按钮**

4. **重复添加2-3个测试数据**，例如：
   - HG RX-78-2 高达（日本定价1500日元，中国市场价格80元）
   - RG RX-93 ν高达（日本定价4500日元，中国市场价格280元）

## 🧪 第二步：测试各项功能

### 1. 查看高达列表

- 点击导航栏的"高达列表"
- 查看所有已添加的高达
- 尝试使用搜索功能
- 尝试按级别筛选（MG、HG、RG等）

### 2. 查看"算"数计算

- 在列表页面可以看到每个高达的"算"数
- 点击某个高达的"详情"按钮
- 在详情页面查看详细的价格信息和"算"数计算

**"算"数说明：**
- 算 = (中国市场价格 / 日本定价) × 100
- 例如：1000日元定价，卖100元 = 10算
- 数值越小表示性价比越高

### 3. 测试想要列表

- 点击某个高达的"详情"按钮
- 点击"添加到想要列表"按钮
- 点击导航栏的"想要列表"查看
- 可以移除不想要的项目

### 4. 测试已购买列表

- 在高达详情页面，点击"添加到已购买"按钮
- 点击导航栏的"已购买"查看已购买的高达
- 可以记录购买价格、日期、平台等信息（需要编辑功能，后续完善）

### 5. 测试优惠券功能

#### 添加优惠券：

1. 点击导航栏的"优惠券"
2. 点击"添加优惠券"按钮
3. 填写优惠券信息，例如：
   - 平台：`拼多多`
   - 折扣类型：`百分比折扣`
   - 折扣值：`8`（表示8折，即20%折扣）
   - 最高减免：`50`（表示最高减50元）
   - 有效期：设置一个未来的日期

4. 点击"保存"

#### 分析优惠券：

1. 在优惠券列表页面，点击某个优惠券的"分析"按钮
2. 系统会分析想要列表中的哪些高达最适合使用这个优惠券
3. 按节省金额排序显示

## 🎯 第三步：开始实际使用

现在您已经熟悉了基本功能，可以：

1. **录入真实的高达数据**
   - 添加您关心的高达信息
   - 记录价格信息

2. **管理收藏**
   - 将想要购买的添加到"想要列表"
   - 记录已购买的到"已购买列表"

3. **分析优惠**
   - 添加优惠券
   - 分析哪些高达使用优惠券最划算

## 🔧 常见操作

### 停止应用

在终端窗口中按 `Ctrl+C` 可以停止应用

### 重新启动应用

1. 停止应用（Ctrl+C）
2. 重新运行：`py app.py`

### 数据库

- 数据库文件：`gunpla.db`
- 数据会自动保存
- 可以备份这个文件

## ⚠️ 注意事项

1. **保持终端窗口打开**
   - 应用运行期间，终端窗口必须保持打开
   - 关闭终端窗口会停止应用

2. **汇率设置**
   - 默认汇率：1人民币 = 20日元
   - 可以在 `config.py` 中修改
