# 如何使用应用

## 重要说明

**Flask是一个Web服务器，不会自动打开浏览器窗口！**

运行 `python app.py` 后，应用会在后台运行，您需要**手动打开浏览器**来访问应用。

## 使用步骤

### 1. 运行应用

在终端中运行：
```bash
python app.py
```

您应该看到类似这样的输出：
```
 * Serving Flask app 'app'
 * Debug mode: on
WARNING: This is a development server. Do not use it in a production deployment. Use a production WSGI server instead.
 * Running on http://127.0.0.1:5000
 * Running on http://0.0.0.0:5000
Press CTRL+C to quit
```

### 2. 打开浏览器

看到上面的输出后，打开您的浏览器（Chrome、Edge、Firefox等）

### 3. 访问应用

在浏览器地址栏输入以下地址之一：

- **http://localhost:5000**
- **http://127.0.0.1:5000**

然后按回车键。

### 4. 看到页面

如果一切正常，您应该看到应用的首页！

## 如果看不到页面

### 检查1：应用是否在运行？

查看终端窗口，应该看到 "Running on http://..." 的提示。
如果没有，可能有错误信息。

### 检查2：端口是否被占用？

如果看到端口被占用的错误，可以：
1. 关闭占用5000端口的其他程序
2. 或者修改 `app.py` 中的端口号

### 检查3：是否有错误信息？

查看终端中的错误信息，常见问题：
- 缺少依赖包
- 数据库文件权限问题
- 模板文件路径问题

## 停止应用

在运行应用的终端窗口中，按 `Ctrl+C` 可以停止应用。

## 提示

- 应用运行期间，终端窗口必须保持打开
- 关闭终端窗口会停止应用
- 如果需要同时使用终端做其他事情，可以打开新的终端窗口
