### 示例：使用实时汇率

可以添加实时汇率获取功能：

```python
import requests

def get_exchange_rates():
    """获取实时汇率（示例，需要实际的汇率API）"""
    try:
        # 使用免费的汇率API，例如 exchangerate-api.com
        response = requests.get('https://api.exchangerate-api.com/v4/latest/JPY')
        data = response.json()
        return {
            'jpy_to_usd': data['rates']['USD'],
            'jpy_to_cny': data['rates']['CNY']
        }
    except:
        # 如果API失败，使用默认值
        return {
            'jpy_to_usd': 0.0067,
            'jpy_to_cny': 0.05
        }
```

然后在 `scrape_rg_series` 函数中使用：

```python
rates = get_exchange_rates()
converted = scraper.convert_price(
    detail['price_jp_msrp'],
    jpy_to_usd=rates['jpy_to_usd'],
    jpy_to_cny=rates['jpy_to_cny']
)
```

## 📝 汇率参考

### 历史汇率范围（仅供参考）

**日元对美元（JPY/USD）：**
- 2020-2024年：约 0.0065 - 0.0075（130-150 JPY = 1 USD）

**日元对人民币（JPY/CNY）：**
- 2020-2024年：约 0.045 - 0.055（18-22 JPY = 1 CNY）

### 当前常用汇率（2024年）

- **1 USD ≈ 150 JPY**
- **1 CNY ≈ 20 JPY**

## ⚠️ 注意事项

1. **汇率会波动**：实际汇率每天都会变化
2. **定价不等于市场价**：换算的是官方定价（MSRP），实际市场价格可能不同
3. **地区差异**：不同地区的定价策略可能不同，换算仅供参考

## 💡 建议

- 如果只是用于参考，使用固定汇率即可
- 如果需要精确计算，建议使用实时汇率API
- 可以在配置文件中设置汇率，方便随时更新
