# 开始爬取 - 快速指南

## ✅ 已配置完成

脚本已配置为使用方法1：爬取指定的级别

**当前配置：**
- 爬取级别：RG
- 包含价格：是
- 自动识别子分类：是

## 🚀 运行命令

```bash
cd gunpla_price_tool
py 78dm_auto_scraper.py
```

## 📝 爬取过程

脚本会：

1. **自动查找级别链接**
   - 访问资料库
   - 找到RG级别的链接

2. **爬取RG系列**
   - 爬取所有模型列表
   - 自动识别子分类（普通版、网络限定版等）
   - 显示子分类统计

3. **爬取价格信息**
   - 访问每个模型的详情页面
   - 提取日元定价
   - 自动换算为美元和人民币定价

4. **保存到数据库**
   - 自动去重
   - 更新已有数据
   - 显示保存结果

## ⏱️ 预计时间

- **RG级别（约219个模型）**：约 5-6 分钟
